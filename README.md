# Procesarea Semnalelor

Laborator pentru curs de procesarea semnalelor.

## Structură

- `lab1/` - Semnale continue și discrete, eșantionare
- `lab2/` - Semnale audio, zgomot, aproximări
- `lab3/` - Transformata Fourier, matrici DFT

## Rulare

```bash
python lab1/lab1_semnale.py
python lab2/lab2_semnale.py
python lab3/lab3.py
```

pip install numpy matplotlib scipy sounddevice
